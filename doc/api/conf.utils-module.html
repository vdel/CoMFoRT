select_mode()

source code 

Permet de mettre confstatic.mode à jour, avec deux valeurs possibles :

  • local : on est sur le mini serveur en Python
  • dynamic : on est sur un vrai serveur, genre Apache

latex_colors(chdir=None)

source code 

De quoi récupérer un couple contenant #123456,#123456 représentant les couleurs d'arrière-plan et d'avant-plan pour générer du latex. Renvoie blanc et noir sinon

Parameters:
  • chdir - facultatif, si présent, on change au dossier en question pour être de nouveau dans src. La fonction retourne en revenant au dossier d'où elle a été appelée
